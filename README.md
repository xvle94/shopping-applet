# wxapp-mall
微信小程序 商城（https://github.com/）


## 功能
- [x] 首页——index
- [x] 商城——mall
- [x] 搜索——search
- [x] 消息——msg
- [x] 购物车——cart
- [x] 我的——user
- [x] 商品列表——category
- [x] 商品详情——details
- [x] 订单——order
- [x] 地址管理——address
- [x] 活动——active
- [x] 活动详情——activeDetail
- [x] 店铺——shop
- [x] 店铺详情——list
- [x] 积分商城——integral
- [x] 停车缴费——parking
- [x] 停车支付——parkPay
- [x] 停车历史——parkHistory
